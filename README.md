# Bot discord basic

## Mô tả
Bot Discord được viết bằng Python, có hỗ trợ MongoDB và kiến trúc mở rộng với Cogs, Events, Utils.

## Cài đặt
```bash
git clone https://github.com/huuvuong0123/Bot-discord-basic.git
cd Bot-discord-basic
pip install -r requirements.txt
```

## Chạy code
```bash
python bot.py
```
